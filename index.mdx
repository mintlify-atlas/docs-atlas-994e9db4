Explorador de Archivos is a Windows desktop application that goes far beyond a standard file browser. Built with C# and WinForms on .NET 8, it integrates a complete media player, photo editor, screen recorder, multi-format document editor, and a powerful data analysis suite — all accessible from a single, unified window.

## Getting Started

<Steps>
  <Step title="Build or download the application">
    Open `ExploradorDeArchivos.sln` in Visual Studio 2022 and publish for Windows x64 with .NET 8. See the [Installation](/installation) page for full prerequisites.
  </Step>
  <Step title="Launch and browse your files">
    Run the executable. The app opens to your Documents folder. Use the left panel for quick access to Downloads, Pictures, Music, Videos, Desktop, and connected drives.
  </Step>
  <Step title="Open a media file">
    Double-click any audio file (`.mp3`, `.wav`, `.aac`) to open the MP3 player, or a video file (`.mp4`, `.mkv`, `.avi`) to launch the video player automatically.
  </Step>
  <Step title="Explore the data tools">
    Click **Data Fusion** in the toolbar to open the Data Fusion Arena, then import a CSV or connect to a database to generate charts and run analysis.
  </Step>
</Steps>
